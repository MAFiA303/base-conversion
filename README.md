# base-conversion
change numbers between different bases
```python
newbase(number,old base, new base)
```
###### example: Change number 303 from base 10 to base 2

```python
newBase(303,10,2)
#output is a string:
'100101111'
```

###### use minBase to find the minimum base the input can be at, ex:
```python
minBase('saad')
29
newBase('saad',29,22)
'2kkkd'
```
